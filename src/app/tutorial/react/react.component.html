React Example

Step 01:

  1. Install Nodejs on your machine
  2. After that we need to check nodejs is installed or not
  3. open your command prompt type below give command
  4. Command is "node --version"
  5. In meantime npm installed with nodejs
  6. use this command to check the npm version npm --version

Step 02:

  1. How to create an React Project in your Machine
  2. Open command prompt in your machine at spacific folder, use this command to create a project "npx create-react-app projectname"
  3. Go to your current project directory cd "projectname"
  4. How to run React project use below command.
  5. Command is "npm start"
  6. It will automatically run on your default browser

Step 03:

  1. How to add Bootstrap in your project
  2. Open current directory on your command prompt use this command"npm install bootstrap"
  3. How to check bootstrap is installed or not? Check on your package.json file on your project.

Step 04:

  1. We need to add dependency for react routing
  2. Open current directory on your command prompt use this command"npm install react-router-dom"
  3. How to check react-router-dom is installed or not? Check on your package.json file on your project.
